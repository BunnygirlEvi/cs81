Welcome to the Morse Code Translator! This was made by Evelin Boghozyan.
You can type words or morse code into the text area, and you can push the play button to hear the morse code, and the stop button to stop it.
There is a game function where you are asked to translate morse code into words, and there are 5 possible questions that can be asked.
If playing the game, please type your answers in correct grammar without commas and periods.
Every answer is only 1 sentence long.
Use / as spaces and spaces between letters when typing in morse code for the translator.
Symbols and newlines will be ignored.
